3 лабораторная, в которой нужно было сделать симулятор фермы. Я реализовала это создав классы растений, дав каждому из них диапазон значения влажности, в которм они могут расти. Изначально все поле являетя землей, используя механики, его можно сделать водой и посадить растение, в зависимости от условий растение будет либо яркого цвета (оно проросло), либо тусклого (растение завяло).
